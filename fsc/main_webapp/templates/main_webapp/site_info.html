Website info

This website intends to showcase my skillset in terms of web development and machine learning and general purpose programming. The development mainly uses HTML, CSS, Bootstrap and Python. The framework the website runs on is Django. Major Python packages used for backend logic includes but not limited to scikit-learn, nltk, opencv, pandas, numpy, matplotlib. The webiste data is held in a PostgresSQL database.
